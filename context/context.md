# Personal Portfolio Context

## Objective

Build a personal portfolio website for Sanket Patel.

The portfolio is not a project showcase.

The portfolio is a product-engineering profile designed to attract:

* Remote startups
* SaaS companies
* AI-enabled companies
* Product engineering teams
* Technical founders
* Engineering managers

The website should communicate:

"Engineer who can independently own and deliver complex product systems."

The website should not feel like a freelancer portfolio.
